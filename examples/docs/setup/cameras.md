# Camera Setup and Calibration

This guide explains how to install and calibrate the cameras for the example
product. Proper calibration is required before the first scan.

## Mounting the camera

Mount the camera on the supplied bracket and tighten the two screws. Keep the
lens at least 30cm from the target surface.

## Calibrating

Run the calibration wizard from Settings → Devices → Calibrate. Hold the
calibration card flat in the frame until all four corners turn green. The wizard
saves the calibration profile automatically.

## Verifying calibration

After calibration, run a test scan. A good calibration shows alignment error
below 0.2mm in the diagnostics panel.
